## Components

### StreamPlayerCore

.NET library for video streaming using FFmpeg. Provides core functionality for handling video streams.

### StreamPlayerCore.WPF.Control

WPF control for video streaming using FFmpeg. Uses the core library to provide a user interface for video playback in WPF applications.

### StreamPlayerCore.WinForms.Control

WinForms control for video streaming using FFmpeg. Uses the core library to provide a user interface for video playback in WinForms applications.

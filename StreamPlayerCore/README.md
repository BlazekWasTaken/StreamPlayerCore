# StreamPlayerCore

[![NuGet](https://img.shields.io/badge/NuGet-StreamPlayerCore-004880?logo=nuget&logoColor=fff)](https://www.nuget.org/packages/StreamPlayerCore/)

StreamPlayerCore is a lightweight and efficient media streaming library designed for developers who want to integrate
video streaming capabilities into their applications. It supports the RTSP protocol.

Built with .NET 10.

## Installation

You can install the StreamPlayerCore library via NuGet Package Manager:

```
Install-Package StreamPlayerCore
```

## Usage

For a complete example of using StreamPlayerCore, please refer to
the [StreamPlayerCore.WinForms.Control](https://github.com/BlazekWasTaken/StreamPlayerCore/blob/2.1.4/StreamPlayerCore.WinForms.Control/README.md)
and [StreamPlayerCore.WPF.Control](https://github.com/BlazekWasTaken/StreamPlayerCore/blob/2.1.4/StreamPlayerCore.WPF.Control/README.md) projects.

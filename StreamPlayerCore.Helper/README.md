# StreamPlayerCore.Helper

[![NuGet](https://img.shields.io/badge/NuGet-StreamPlayerCore.Helper-004880?logo=nuget&logoColor=fff)](https://www.nuget.org/packages/StreamPlayerCore.Helper/)

A supplementary library for StreamPlayerCore with common helper functions and utilities.

Built with .NET 10.

## Installation

You can install the StreamPlayerCore.Helper library via NuGet Package Manager:

```
Install-Package StreamPlayerCore.Helper
```

## Usage

For usage examples, please refer to the [StreamPlayerCore.WinForms.Control](../StreamPlayerCore.WinForms.Control) and [StreamPlayerCore.WPF.Control](../StreamPlayerCore.WPF.Control) projects.
